# Incident Reports

This folder contains detailed reports of security incidents identified during attack simulations within the Enterprise SOC Homelab.

Each report documents the investigation process, indicators of compromise (IOCs), MITRE ATT&CK techniques, response actions, and lessons learned.

## Report Template

- Executive Summary
- Timeline
- Detection
- Investigation
- Indicators of Compromise
- MITRE ATT&CK Mapping
- Containment
- Recovery
- Lessons Learned
