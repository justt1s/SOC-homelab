# Enterprise SOC Homelab

## Overview

This project demonstrates the deployment and operation of a Security Operations Center (SOC) built in VMware.

## Technologies

- Wazuh
- TheHive
- Cortex
- Shuffle
- IRIS
- Snort
- Splunk (Learning)
- Kali Linux
- Ubuntu
- Windows Server
- Windows 11

## Objectives

- Detect cyber attacks
- Investigate incidents
- Automate incident response
- Practice threat hunting
- Learn enterprise SOC workflows
