# SOAR

This folder contains Security Orchestration, Automation, and Response (SOAR) workflows built using Shuffle.

## Planned Workflows

- IOC Enrichment
- VirusTotal Lookup
- AbuseIPDB Lookup
- TheHive Case Creation
- Email Notification
- Alert Triage
- Automated Incident Response

## Objectives

- Reduce manual investigation time
- Automate repetitive SOC tasks
- Improve incident response efficiency
