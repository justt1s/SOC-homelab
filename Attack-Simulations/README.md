# Attack Simulations

This folder contains simulated cyber attacks performed in the homelab to validate detection capabilities, improve incident response skills, and evaluate security controls.

## Planned Simulations

- Brute Force Attack
- Port Scanning
- PowerShell Execution
- Mimikatz Credential Dumping
- Reverse Shell
- Ransomware Simulation
- Persistence Techniques
- Privilege Escalation
- Phishing Simulation
- Web Exploitation

## Objectives

- Generate realistic security events
- Validate SIEM detections
- Practice incident investigation
- Improve SOC workflows
